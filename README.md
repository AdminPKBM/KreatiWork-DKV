# KreatiWork-DKV
KreatiWork DKV adalah platform media pembelajaran digital mandiri (offline-friendly) yang dirancang khusus untuk mendukung proses belajar mengajar siswa SMK Program Keahlian Desain Komunikasi Visual (DKV), khususnya dalam menyelaraskan kompetensi dengan kebutuhan kurikulum SMK Pusat Keunggulan.
